---
title: "PSQL Tool"
---

<div id="psql_tool" class="registered_link"></div>

PSQL tool allows user to connect to PostgreSQL/EDB Advanced server using psql terminal.

-   Open PSQL Tool from Tools menu or PSQL tool button from browser tree or from context menu.
-   PSQL will connect to the current connected database from browser tree.

<img src="../images/psql_tool.png" class="align-center" alt="PSQL tool window" />

You can open multiple instance of the PSQL tool in individual tabs simultaneously. To close the PSQL tool, click the *X* in the upper-right hand corner of the tab bar.

**Note:** This feature is not available for Windows Platforms.
