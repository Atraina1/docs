---
title: "Prerequisites for managing Barman"
---

Before adding a Barman server to the PEM console,

- You must manually install and configure Barman on the Barman host. For more information about installing and configuring Barman, please see the ``Barman docs`` available at:

    https://www.enterprisedb.com/docs/supported-open-source/barman/

- Install ``pg-backup-api`` tool on Barman host.
