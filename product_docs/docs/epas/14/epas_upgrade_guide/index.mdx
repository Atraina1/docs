*EDB Postgres Advanced Server Upgrade* is a comprehensive guide about upgrading EDB Postgres Advanced Server (Advanced Server). In this guide you will find detailed information about using:

-   `pg_upgrade` to upgrade from an earlier version of Advanced Server to Advanced Server 14.
-   `yum` to perform a minor version upgrade on a Linux host.
-   `StackBuilder Plus` to perform a minor version upgrade on a Windows host.
