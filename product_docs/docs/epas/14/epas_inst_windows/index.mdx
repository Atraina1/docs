The *EDB Postgres Advanced Server Installation Guide* is a comprehensive guide to installing EDB Postgres Advanced Server (EDB Postgres Advanced Server). In this guide you will find detailed information about:

-   Graphical installation options available through the interactive setup wizard on Windows.
-   Managing an EDB Postgres Advanced Server installation.
-   Configuring an EDB Postgres Advanced Server package installation.
-   Uninstalling EDB Postgres Advanced Server and its components.
