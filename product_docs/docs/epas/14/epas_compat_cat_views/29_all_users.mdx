title: "ALL_USERS"

<div id="all_users" class="registered_link"></div>

The `ALL_USERS` view provides information on all user names.

| Name            | Type   | Description                                                                                                                                                                                                                                                   |
| --------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `username`      | `TEXT` | Name of the user.                                                                                                                                                                                                                                             |
| `user_id`       | `OID`  | Numeric user id assigned to the user.                                                                                                                                                                                                                         |
| `created`       | `DATE` | Timestamp for the creation of an object.                                                                                                                                                                                                                      |
| `last_ddl_time` | `DATE` | Timestamp for the last modification of an object resulting from a DDL statement including grants and revokes. <br /> The `edb_last_ddl_time` and `edb_last_ddl_time_shared` catalogs stores the creation time and `last_ddl_time` for all the objects. <br /> |
