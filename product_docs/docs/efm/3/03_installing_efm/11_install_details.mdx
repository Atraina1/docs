---
title: "Installation Details"

---



Components are installed in the following locations:

| Component                         | Location                    |
| --------------------------------- | --------------------------- |
| Executables                       | `/usr/edb/efm-<4.4>/bin`        |
| Libraries                         | `/usr/edb/efm-<4.4>/lib`        |
| Cluster configuration files       | `/etc/edb/efm-<4.4>`            |
| Logs                              | `/var/log/efm-<4.4>`           |
| Lock files                        | `/var/lock/efm-<4.4>`           |
| Log rotation file                 | `/etc/logrotate.d/efm-<4.4>`    |
| sudo configuration file           | `/etc/sudoers.d/efm-<4.4>`       |
| Binary to access VIP without sudo | `/usr/edb/efm-<4.4>/bin/secure` |
