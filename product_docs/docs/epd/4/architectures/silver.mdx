The AlwaysOn Silver architecture includes the following:

-  Three BDR data nodes
-  Two HARP-Proxy nodes
-  Barman offsite (offsite is optional but recommended)


BDR and HARP nodes should be spread across AZs, and Barman can be located in location A.
