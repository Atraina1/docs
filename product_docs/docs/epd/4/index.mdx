EDB Distributed Postgres is a technology that let's you build
distributed PostgreSQL setup, both for the purpose
of continuous availability and for better scalability.

The EDB Distributed Postgres can be used both with EDB Advanced Postgres Server
as well as with open source PostgreSQL.
