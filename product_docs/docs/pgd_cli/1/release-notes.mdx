## Release 1.0.0 (2022-05-17)

This is an initial release for PGD Command Line Interface (CLI). PGD CLI is a tool to manage your BDR cluster. It allows you to run commands against BDR clusters. You can use it to inspect and manage cluster resources, and view membership events, worker errors etc.

Following commands are available with initial release -

- `pgd check-health` - Checks the health of the BDR cluster (BDR-2015)
- `pgd show-camo` - Shows BDR CAMO (Commit at Most Once) details (BDR-2069)
- `pgd show-clockskew` - Shows the status of clock skew between each BDR node pair (BDR-1987)
- `pgd show-events` - Shows background worker errors, node membership changes etc. (BDR-1988)
- `pgd show-nodes` - Shows all nodes in the BDR cluster and their summary (BDR-2088)
- `pgd show-raft` - Shows BDR Raft (consensus protocol) details (BDR-1986)
- `pgd show-replslots` - Shows the status of BDR replication slots (BDR-1985)
- `pgd show-subscriptions` - Shows BDR subscription (incoming replication) details (BDR-1985)
- `pgd show-version` - Shows the version of BDR and Postgres installed on each node (BDR-2025)
