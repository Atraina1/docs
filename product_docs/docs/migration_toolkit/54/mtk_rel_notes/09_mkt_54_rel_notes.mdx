New features, enhancements, bug fixes, and other changes in Migration Toolkit 55.0.0 include:

| Type | Description |
| ---- |------------ |
| Enhancement | Support for Postgres and EDB Postgres Advanced Server version 13. |
| Enhancement | Support for Ubuntu version 20. |
| Enhancement | New Windows 64-bit installer. **Note:** If you have an existing 32 bit based Migration Toolkit installation, you must uninstall and reinstall the Migration Toolkit using the new Windows 64 bit installer.
