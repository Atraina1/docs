New features, enhancements, bug fixes, and other changes in the EDB JDBC Connector `42.3.2.1` include:

| Type           | Description                                                                                                                                                                                                                                                                                                                |
| -------------- | ------------------------------------------------------------------------------- |
| Upstream Merge | Merged with the upstream community driver version `42.3.3`. See the community [JDBC documentation](https://jdbc.postgresql.org/documentation/changelog.html#version_42.3.3) for details.                                                                                                                                   |
| Security Fix   | GHSA-673j-qm5f-xpv8: The driver logging is configured only using `java.util.logging.conifguration` mechanisms such as `logging.properties`. |                                                                                                                                                                                                                                                                     |
