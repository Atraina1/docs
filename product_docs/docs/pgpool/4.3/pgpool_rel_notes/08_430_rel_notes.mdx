Pgpool-II 4.3.0 includes the following upstream merge and updates:

| Type           | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Upstream Merge | Merged with community Pgpool-II 4.3.0. See the community [Release Notes](https://www.pgpool.net/docs/43/en/html/release-4-3-0.html#bug-fixes) for details.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Security Fix   | Reject extraneous data after SSL encryption handshake.<br /><br />In the server-side implementation of SSL negotiation, it was possible for a man-in-the-middle attacker to inject arbitrary SQL commands if it was configured to use cert authentication or hostssl + trust. This resembles PostgreSQL's CVE-2021-23214.<br /> <br /> In the client-side implementation of SSL negotiation, it was possible for a man-in-the-middle attacker to inject arbitrary responses if the database server is using trust authentication with a clientcert requirement. It is not possible with cert authentication because Pgpool-II does not implement the cert authentication between Pgpool-II and PostgreSQL. This resembles PostgreSQL's CVE-2021-23222. |

!!!note
This security fix is also available in EDB Pgpool-II 4.1.9, 4.0.16, 3.7.21, and 3.6.28.
