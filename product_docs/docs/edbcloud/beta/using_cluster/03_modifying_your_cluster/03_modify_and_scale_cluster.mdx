1. Sign in to the [EDB Cloud](https://portal.edbcloud.com) portal.

1. From the [**Clusters**](https://portal.edbcloud.com/clusters) page, select the name of the cluster you want to edit.

2. Select **Edit Cluster** from the top right corner of the **Cluster Info** panel.
3. Change the clusterâ€™s scaling options to the desired values. The following options are available to edit on the corresponding tab:

   | Scaling Option | Tab | 
   | ------------- | ----- |
   | Increase compute power and memory | Instance Type* section of [Operational Settings](../../getting_started/03_create_cluster/#operational_settings) |
   | Increase the number of database connections | [Database Configuration Parameters](05_db_configuration_parameters)
   | Enable high availability | [Availability](../../getting_started/03_create_cluster/#availability) |
