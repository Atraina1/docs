## Synopsis

Shows the credential information for the current context. Use the `set-context-credential` command to assign a default credential on a per-module 
basis.  If this command has an assigned context credential, it is automatically 
be used by all commands, so you don't have to specify the 
`--credential` flag.

```
biganimal get-context-credential [flags]
```
### Flags

```
  -h, --help   help for get-context-credential
```

## Examples

```
$ biganimal get-context-credential
```
