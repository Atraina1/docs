## Synopsis

Updates a Postgres database cluster.

```
biganimal update-cluster [flags]
```

### Flags

```
  -c, --credential string      credential
  -h, --help                   help for update-cluster
  -H, --high-availability      New value for High Availability flag
  -i, --id string              Cluster ID
  -I, --instance-type string   New Instance Type ID
  -n, --name string            Existing Cluster name
  -N, --new-name string        New Cluster name
  -W, --password string        New Cluster password
  -P, --private-networking     New value for Private Networking flag
  -p, --provider string        Provider ID
  -r, --region string          Region ID
```

## Examples

```
$ biganimal update-cluster --id p-c5fh47nfjcg4ci71uv32 --new-name some_updated_name --private-networking=false --high-availability=true
$ biganimal update-cluster --name existing_name --provider azure --region francecentral --new-name some_updated_name
$ biganimal update-cluster --name existing_name --provider azure --region francecentral --high-availability=true
$ biganimal update-cluster --name existing_name --provider azure --region francecentral --new-name some_updated_name --private-networking=false
$ biganimal update-cluster --name existing_name --provider azure --region francecentral --new-name some_updated_name --private-networking=false --high-availability=true
```
