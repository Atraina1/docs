## Synopsis

Shows all instance types.

```
biganimal show-instance-types [flags]
```


### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-instance-types
  -p, --provider string     Provider ID
  -r, --region string       Region ID
```
## Examples

```
$ biganimal show-instance-types --provider azure --region westus2
```
