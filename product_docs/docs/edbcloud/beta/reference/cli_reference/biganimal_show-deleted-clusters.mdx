## Synopsis

Shows all deleted clusters or a subset you specify using the `--name`, `--provider`, or `--region` flags.

```
biganimal show-deleted-clusters [flags]
```
### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-deleted-clusters
  -i, --id string           Cluster ID
  -n, --name string         Cluster Name
  -r, --region string       Region ID
```

## Examples

```
$ biganimal show-deleted-clusters
$ biganimal show-deleted-clusters --id p-c5fh47nfjcg4ci71uv32
$ biganimal show-deleted-clusters --name some_cluster_name
$ biganimal show-deleted-clusters --name db_cluster --region-id westus2
```
