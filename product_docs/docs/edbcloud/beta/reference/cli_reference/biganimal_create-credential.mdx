## Synopsis

Creates a new credential.

```
biganimal create-credential [flags]
```

### Flags

```
  -a, --address string   credential host address
  -h, --help             help for create-credential
  -n, --name string      credential name
  -o, --port string      credential host port
```
## Examples

```
$ biganimal create-credential --name cred1 --address portal.biganimal.com --port 443
$ biganimal create-credential -n cred1 -a portal.biganimal.com -o 443
```
