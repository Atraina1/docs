## Synopsis

Shows all volume properties.

```
biganimal show-volume-properties [flags]
```

### Flags

```
  -c, --credential string    credential
  -h, --help                 help for show-volume-properties
  -p, --provider string      Provider ID
  -r, --region string        Region ID
  -v, --volume-type string   Volume Type ID
```

## Examples

```
$ biganimal show-volume-properties --provider azure --region westus2 --volume-type azurepremiumstorage
```
