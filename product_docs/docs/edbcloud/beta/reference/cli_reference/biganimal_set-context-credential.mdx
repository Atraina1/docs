## Synopsis

Sets the default credential. When you have set a default credential, you can use commands that require a credential without having to pass it. The commands use the default credential. 


```
biganimal set-context-credential
```
### Flags

```
  -h, --help   help for set-context-credential
```

### Examples

```
$ biganimal set-context-credential <credentials_name>
```
