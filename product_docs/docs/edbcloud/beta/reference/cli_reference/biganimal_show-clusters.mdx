## Synopsis

Shows all Postgres clusters or the subset you specify using the `--provider` and `--region` flags.

```
biganimal show-clusters [flags]
```

### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-clusters
  -i, --id string           Cluster ID
  -n, --name string         Cluster Name
  -p, --provider string     Provider ID
  -r, --region string       Region ID
```
## Examples

```
$ biganimal show-clusters
$ biganimal show-clusters --id p-c5fh47nfjcg4ci71uv32
$ biganimal show-clusters --provider azure --region eastus2
$ biganimal show-clusters --name cli-db --provider azure --region eastus2
```
