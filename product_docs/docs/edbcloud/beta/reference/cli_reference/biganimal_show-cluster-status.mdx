## Synopsis

Shows the current status of the specified cluster.

```
biganimal show-cluster-status [flags]
```

### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-cluster-status
  -i, --id string           Cluster ID
  -n, --name string         Cluster Name
  -p, --provider string     Provider ID
  -r, --region string       Region ID
```

## Examples

```
$ biganimal show-cluster-status --id p-c5fh47nfjcg4ci71uv32
$ biganimal show-cluster-status --name cli-db --provider azure --region eastus2
```
