Enable confirmation when creating/deleting/updating resources

## Synopsis

Enables user confirmation.  When enabled, the CLI 
asks the user for confirmation before creating, deleting, or updating resources.

```
biganimal enable-confirm [flags]
```

### Flags

```
  -h, --help   help for enable-confirm
```
