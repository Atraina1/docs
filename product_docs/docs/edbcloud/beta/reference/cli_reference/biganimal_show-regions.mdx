## Synopsis

Shows all available regions.

```
biganimal show-regions [flags]
```

### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-regions
  -p, --provider string     Provider ID
```

## Examples

```
$ biganimal show-regions --provider azure
```
