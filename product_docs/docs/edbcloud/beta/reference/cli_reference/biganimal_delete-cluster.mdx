## Synopsis

Deletes the specified Postgres cluster.

```
biganimal delete-cluster [flags]
```

### Flags

```
  -c, --credential string   credential
  -h, --help                help for delete-cluster
  -i, --id string           Cluster ID
  -n, --name string         Cluster Name
  -p, --provider string     Provider ID
  -r, --region string       Region ID
```

## Examples

```
$ biganimal delete-cluster --id p-c5fh47nfjcg4ci71uv32
$ biganimal delete-cluster --name cli-db --provider azure --region eastus2
```
