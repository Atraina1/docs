## Synopsis

Shows all available volume types.

```
biganimal show-volume-types [flags]
```

### Flags

```
  -c, --credential string   credential
  -h, --help                help for show-volume-types
  -p, --provider string     Provider ID
  -r, --region string       Region ID
```

## Examples

```
$ biganimal show-volume-types --provider azure --region westus2
```
