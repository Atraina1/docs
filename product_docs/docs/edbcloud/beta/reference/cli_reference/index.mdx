Use the CLI to create and manage database 
cluster resources and events.  

Before using the CLI, you must create 
a credential using the [create-credential](biganimal_create-credential) command. 



### See also

-   [create-cluster](biganimal_create-cluster)	 - Create cluster
-   [create-credential](biganimal_create-credential)	 - Create credentials

-   [disable-confirm](biganimal_disable-confirm)	 - Disable confirmation before creating/deleting/updating resources
-   [disable-prompt](biganimal_disable-prompt)	 - Disable prompting for missing required parameters
-   [enable-confirm](biganimal_enable-confirm)	 - Enable confirmation when creating/deleting/updating resources
-   [enable-prompt](biganimal_enable-prompt)	 - Enable prompting for missing required parameters
-   [get-confirm](biganimal_get-confirm)	 - Get the current value for the confirmation mode
-   [get-context-credential](biganimal_get-context-credential)	 - Show the credential information for current context

-   [get-prompt](biganimal_get-prompt)	 - Get the current value for the prompt mode
-   [reset-credential](biganimal_reset-credential)	 - Reset credential
