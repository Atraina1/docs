## Synopsis

Creates a Postgresql database cluster in interactive mode or from a cluster configuration file.


```
biganimal create-cluster [flags]
```


### Flags

```
  -f, --clusterConfigFile string   Path to file cluster config yaml file, use 'create-cluster --help' for config file specification
  -c, --credential string          credential
  -h, --help                       help for create-cluster
```
#### Cluster configuration file

The cluster configuration file is in yaml format and follows the following sample:

```
---
clusterName: biganimal_cluster		# <string:	cluster name>
instanceType:
  id: azure:Standard_E2s_v3		# <string:	instance type id>
  ram: 16				# <int:		RAM associated with selected instance type id>
  cpu: 2				# <int:		CPU associated with selected instance type id>
  instanceType: E2s v3			# <string:	instance type name>
password: mypassword@123		# <string:	cluster password (must be at least 12 characters)>
pgConfigMap: []				# <reserved field, default is blank>
postgresType:
  id: postgres				# <string:	postgres type id>
postgresVersion: '13'			# <string:	postgres version>
privateNetworking: false		# <bool:	in private network or public network>
provider:
  cloudProviderId: azure		# <string:	cloud provider id>
region:
  regionId: francecentral		# <string:	provider region id>
replicas: 3				# <int:		replica count>
volumeProperties: P1			# <string:	volume properties id>
volumeType:
  id: azurepremiumstorage		# <string:	volume type id>
  storageClass: managed-premium		# <string:	storage class_ ype>
zoneRedundantHa: true			# <bool:	turn on high-availability>
---
```
## Examples

```
$ biganimal create-cluster
$ biganimal create-cluster --clusterConfigFile path_to_config_file.yaml
```
