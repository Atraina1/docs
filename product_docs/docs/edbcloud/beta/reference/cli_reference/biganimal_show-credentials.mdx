## Synopsis

Shows all credentials or a specific credential if you use the `--name` flag.

```
biganimal show-credentials [flags]
```



### Flags

```
  -h, --help               help for show-credentials
  -n, --name <cred_name>   credential name
```
## Examples

```
$ biganimal show-credentials
```
