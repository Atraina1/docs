## Prerequisites

To install MySQl Foreign Data Wrapper, you must have:

- EDB Postgres Advanced Server installed. See [Installing EDB Postgres Advanced Server](epas/14/epas_inst_linux/01_epas_using_edb_repository/02a_edb-postgres-advanced-server_14_rhel-7-or-centos-7-or-ol-8_x86_64).
- Access to the EDB repository. See [Setting up the Repository](/epas/latest/epas_inst_linux/01_epas_using_edb_repository/edb-postgres-advanced-server_14_rhel-8-or-centos-8-or-ol-8_x86_64#setting-up-the-repository).
- MySQL installed. See the platform specific instructions below. 


## RHEL 7/CentOS 7 on x86-64 Install Command

```text
# Before installing MySQL FDW, download and install the MySQL repo using the following commands:
yum -y install https://dev.mysql.com/get/mysql80-community-release-el7-3.noarch.rpm

# Enable the appropriate subrepository and install the MySQL Foreign data Wrapper.

# For MySQL 8:
yum -y install --enablerepo=mysql80-community --disablerepo=mysql57-community edb-as<xx>-mysql8_fdw

# For MySQL 5:
yum -y install --enablerepo=mysql57-community --disablerepo=mysql80-community edb-as<xx>-mysql5_fdw
```
where `<xx>` is the EDB Postres Advanced Server version number.



#### Troubleshooting the installation

If the package was not found, you may not have set up access to the repo. See [Setting up the Repository](/epas/latest/epas_inst_linux/01_epas_using_edb_repository/edb-postgres-advanced-server_14_rhel-7-or-centos-7-or-ol-7_x86_64#setting-up-the-repository).

If yum asks for your permission to import the key to your local server, it means the package is signed by a source that is not recognized by your system. If you are satisfied that the packages come from a trustworthy source when prompted, enter `y`, and press `Return` to continue.

If yum encounters a dependency that it cannot resolve, it provides a list of the required dependencies that you must manually resolve.
