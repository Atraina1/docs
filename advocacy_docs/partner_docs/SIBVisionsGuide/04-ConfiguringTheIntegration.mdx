Implementing VisionX with EDB Postgres Advanced Server and EDB Postgres Extended requires the following components:

- EDB Postgres Advanced Server
- EDB Postgres Extended
- VisionX low-code platform
- Oracle database
- Eclipse with VisionX ePlug 

## Prerequisites

- A running EDB Postgres Advanced Server instance
- A running EDB Postgres Extended instance
- A running Oracle database instance
- VisionX low-code platform installed
- Eclipse with VisionX ePlug installed
- An Oracle Forms or Oracle APEX application
