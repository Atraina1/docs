### Prerequisites
- A running EDB Postgres Advanced (EPAS) instance.

- Flyway Teams Edition installed.

Descriptions of the configuration options that will be set in the flyway.conf file to configure Flyway for EDB Postgres Advanced:

Open the flyway.conf file in the flyway-7.11.4/conf directory and add the following EDB Postgres 
Advanced connection information which will be used by Flyway to connect with EDB Postgres Advanced and perform the migrations.

Following is the description of the configuration options which are set in the flyway.conf.
- <code>flyway.url</code> : Contains the connection string which will be used by flyway to connect with the target 
EPAS database, it contains JDBC driver, Host name/IP address where EPAS database server is running, database port and EPAS user.
- <code>flyway.user</code> : Contains the EPAS database which flyway uses to connect with the database.
- <code>flyway.password</code> : Contains the EPAS database user password.

```bash
flyway.url=jdbc:postgresql://localhost:5460/edb
flyway.user=centos
flyway.password=
```
