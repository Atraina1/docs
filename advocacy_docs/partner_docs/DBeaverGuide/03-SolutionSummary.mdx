title: 'Solution Summary'
description: 'Brief explanation of the solution and its purpose'
---
DBeaver is a tool for managing EDB Postgres Advanced Server. With DBeaver you can manipulate your data like in a regular spreadsheet, create analytical reports based on records from different data storages, export information in an appropriate format. For advanced database users DBeaver suggests a powerful SQL-editor, plenty of administration features, abilities of data and schema migration, monitoring database connection sessions, and a lot more.
