**Implementing DBeaver to work with PostgreSQL databases requires the following components:**

- EDB Postgres Advanced Server (EPAS).
- DBeaver Enterprise Edition.

### Prerequisites

- A running EDB Postgres Advanced Server (EPAS).
- DBeaver Enterprise Edition installed.


The steps below show an example of how to configure DBeaver for EPAS.

### Configuring DBeaver

1. Create a new database connection:

   a. Select the **Database** tab and click on **New Database Connection**

   <p align="center">
        <img  src="Images/DatabaseConnection1.png">
   </p>

   b. Select **EnterpriseDB** from list of available database drivers and click **Next**

   <p align="center">
        <img  src="Images/DatabaseConnection2.png">
   </p>

   c. Enter the corresponding values for **host**, **Database**, **Username**, and **Password** for your database server setup
   
   !!! Note
    Values of **Database** and **Username** in the example below are defaults for EPAS.
  
   <p align="center">
        <img  src="Images/DatabaseConnection3.png">
   </p>
   
   
   d. Click on **Test Connection**. A confirmation screen similar to the example below will pop up if the connection is successful

   <p align="center">
        <img  src="Images/DatabaseConnection4.png">
   </p>
   
   e. Click on **Finish** to complete creating the new database connection

   <p align="center">
        <img  src="Images/DatabaseConnection3.png">
   </p>
   

2.  Verify new database shows up in the **Database Navigator**:

</p>
