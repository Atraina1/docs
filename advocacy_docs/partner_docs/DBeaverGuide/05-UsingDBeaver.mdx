Some common operations for PostgreSQL databases using DBeaver are:

- Listing tables - displays list of tables belonging to the selected schema.
- Viewing table definition - displays the definition of the selected table.
- Viewing table data - displays data stored in the selected table.
- Viewing database diagram - displays the tables and their relationships in the selected schema.
- Reading data - displays the SQL editor with the data read in from the selected table.
- Exporting data - exports data from a selected table to a file.
- Importing data - imports data from a file into the selected table.
- Generating SQL - displays SQL statements generated for the selected table.

### Exporting data

To export table data to a file, perform the steps below:

   1. Select the desired table and click on **Export Data**

   <p align="center">
        <img  src="Images/ExportData1.png">
   </p>

   2.  Select the format of the data to export and click **Next**

   <p align="center">
        <img  src="Images/ExportData2.png">
   </p>

   3.  Skip the next two screens by accepting defaults and clicking **Next**

   4.  On the **Output** screen, enter the destination directory for the export file in the *Directory:** field and click **Next**

   <p align="center">
        <img  src="Images/ExportData5.png">
   </p>

   5.  Click on **Proceed** to confirm the export

   <p align="center">
        <img  src="Images/ExportDataConfirm.png">
   </p>
