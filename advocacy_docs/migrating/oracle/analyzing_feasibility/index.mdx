In this phase of the migration journey, you need to:

-  Determine the platform for the migration, for example, private cloud or public cloud.
-  Understand the compatibility between the source and target databases to ensure that the elements you need are available.
-  Understand the workarounds available to address areas of incompatibility or incomplete compatibility.
