---
title: Incompatibilities
---

A number of incompatibilities between Oracle and EDB Postgres Advanced Server are either not yet addressed or are worth noting because of their frequent use. 

| Oracle Enterprise            | EDB Postgres Advanced Server |
|------------------------------|------------------------------|
| MERGE                        | UPSERT                       | 
| Advanced queuing             | Yes                          |
| Nested procedures/functions  | Yes                          |
| Pipelined functions          | No                           |
| Empty string = NULL          | Empty string = !NULL         |
| Performs many implicit data type conversions such as a number to a string | Requires you to cast a datatype to the other datatype or an error is produced |
